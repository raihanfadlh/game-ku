Nama Perusahaan game-ku
penjelasan gameKu merupakan sebuah web sederhana mengenai pertandingan sepakbola. Apabila diakses akan menampilkan profil,jadwal pertandingan,hasil pertandingan,dan akan menampilkan point dari masing masing team.
Tujuan Tujuan web ini adalah untuk memberitahukan jadawal pertandingan dan hasi point masing masing Team
manfaat web untuk membantu orang lain dalam menemukan jadwal pertandingan serta point masing masing team
